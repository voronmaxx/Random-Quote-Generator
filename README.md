# 📜 Random Quote Generator

Generate random inspirational or educational quotes using multiple programming languages.

---

## 📖 Overview

This repository contains simple implementations of a random quote generator in:

- JavaScript
- Python
- C#

Perfect for beginners learning:

- Arrays and Lists
- Randomization
- Functions
- Console Applications
